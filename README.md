# CODSOFT Web Development Projects

This repository contains web development projects completed as part of the CODSOFT Internship program.
All projects are built using frontend technologies and demonstrate practical implementation of HTML, CSS, and JavaScript.

## Projects List

### 1. Calculator
A simple and interactive calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.
The project uses JavaScript for logic handling and CSS Grid for button layout.

Technologies Used:
- HTML
- CSS
- JavaScript

---

### 2. Online Quiz Maker
An interactive quiz application where users can answer multiple-choice questions and receive instant feedback.
The quiz dynamically loads questions and calculates the final score using JavaScript.

Technologies Used:
- HTML
- CSS
- JavaScript

---

### 3. MY E COMMERCE WEBSITE
A dynamic e-commerce website frontend that allows users to browse products, add items to a shopping cart,
and view total items and price in real time. The project focuses on modern UI design and dynamic behavior.

Technologies Used:
- HTML
- CSS
- JavaScript
